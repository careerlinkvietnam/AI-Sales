# AI-Sales プロジェクト

## 最初に必ず読むこと

**セッション開始時、必ず以下のファイルを読んでから作業を開始すること：**

```
docs/SESSION_HANDOFF.md
```

このファイルには以下が含まれています：
- プロジェクト概要
- 全ルール（1〜17 + サブルール）
- メール作成ルール（フォーマット、件名、署名）
- 企業処理ワークフロー
- 処理済み企業リスト
- スクリプト一覧と使用方法

## 重要なルール（抜粋）

- **ルール2**: 常にCRMで直接確認する（状況は変化する）
- **ルール14-B**: テンプレートは参考例。過去のメール・コールメモ・訪問メモを確認し、お客様に刺さるメールを作成
- **ルール14-C**: メールフォーマット厳守（会社名+担当者名、署名変更禁止）
- **メール作成ルール**: 本文を先に書き、内容に合った件名を作成。件名は `[内容]【キャリアリンク佐藤】`

## 禁止事項

- テンプレートをそのままコピーして使わない
- 署名を勝手に変更しない
- CRMを確認せずに作業しない
- ユーザーの許可なくCRMに書き込まない

## 作業開始手順

1. `docs/SESSION_HANDOFF.md` を読む
2. 現在のタスクを確認
3. ルールに従って作業開始
